### 이 웹페이지 프로젝트는 스팀리뷰 감성분석 웹페이지입니다.
### Steam API, SteamSPY, Youtube API를 이용하여 게임 리뷰, 정보, 태그와 유튜브 내용에 대한 정보를 airflow를 통해 MongoDB에 저장해두었습니다.
### 리뷰는 KoElectra를 통해 긍/부정 감성 분석 및 키워드 추출하여 DB에 저장해두었습니다. 
### 유튜브는 자막을 추출하여 Gemini를 통해 요약 후 DB에 추가 저장하였습니다.
### 이렇게 저장해둔 DB를 기반으로 웹 구현을 진행한 파일입니다.
